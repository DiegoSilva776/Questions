Installation & Usage

These are the 4 installation steps

  • 1 Download the plugin

  • 2 Install the css and js files

  • 3 Add the container that is going to receive the question objects

  • 4 Create Question objects and append to the container



  • 5 Catch the user's answers on the callback function that you passed into the Question objects, and use it in your application
