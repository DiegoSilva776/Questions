Questions

A Js plugin to display Question objects for your evaluation forms

Requirements

  • jQuery - https://cdnjs.cloudflare.com/ajax/libs/jquery/3.2.1/jquery.min.js
  • Mustache - https://cdnjs.cloudflare.com/ajax/libs/mustache.js/2.3.0/mustache.min.js

Installation

These are the 4 installation steps

  • 1 Download the plugin

  • 2 Install the css and js files

  • 3 Add the container that is going to receive the question objects

  • 4 Create Question objects and append to the container
